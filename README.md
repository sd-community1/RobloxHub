# VNDXS HUB SHOOTER

هذا المستودع يحتوي على ملفات متعددة لـ Roblox Executor.

## المحتويات
- `src/main_script.lua`: الكود الرئيسي للهاك.
- `assets/config.json`: ملف إعدادات وهمي.
- `README.md`: هذا الملف.
- `LICENSE`: ملف الترخيص.
- `CONTRIBUTING.md`: دليل المساهمة.

## كيفية الاستخدام
1. انسخ كود التنفيذ من المصدر الموثوق.
2. الصقه في Roblox Executor.
3. اضغط على `Insert` لفتح/إغلاق واجهة المستخدم.
