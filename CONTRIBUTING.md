# دليل المساهمة

نرحب بمساهماتكم!

## الإبلاغ عن الأخطاء
إذا وجدت خطأ، يرجى الإبلاغ عنه عبر فتح "Issue" جديد.

## اقتراح الميزات
لطلب ميزة جديدة، يرجى فتح "Issue" جديد وشرح الميزة بالتفصيل.

## إرسال طلبات السحب (Pull Requests)
1. قم بعمل Fork للمستودع.
2. أنشئ فرعًا جديدًا (Branch) لعملك: `git checkout -b feature/your-feature-name`.
3. قم بتنفيذ التغييرات.
4. قم بعمل Commit للتغييرات: `git commit -m 'feat: Add new feature'`.
5. ارفع التغييرات إلى الفرع: `git push origin feature/your-feature-name`.
6. افتح "Pull Request".
